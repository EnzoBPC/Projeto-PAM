\# Pokedexdb-AppMoveis



Projeto desenvolvido para a disciplina de Desenvolvimento de Aplicativos Móveis. O aplicativo funciona como uma Pokédex que consome dados de uma API externa e permite ao usuário manter uma lista dos seus Pokémon favoritos salva na nuvem.



\## Funcionalidades implementadas



\- Consumo da PokéAPI para carregar dados, tipos e imagens dos Pokémon.

\- Navegação entre telas (Início, Pokédex e Favoritos) utilizando React Navigation.

\- CRUD completo conectado ao Firebase Firestore: é possível salvar um Pokémon nos favoritos, visualizar a lista, editar um apelido e excluir o registro.

\- Animações de interface aplicadas com a biblioteca react-native-animatable.

\- Código estruturado com separação de componentes.



\## Tecnologias utilizadas



\- React Native (Expo)

\- Firebase (Firestore)

\- React Navigation

\- React Native Animatable

\- JavaScript
